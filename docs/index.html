System Design

Architecture Overview

Microservices architecture with GitOps deployment pipeline

Microservices Architecture

Our system implements a modern microservices architecture with clear separation of concerns, enabling independent scaling, deployment, and maintenance of each component.

  • API Gateway for centralized routing and authentication
  • Service mesh for inter-service communication
  • Event-driven architecture with message queuing
  • Distributed tracing and monitoring
  • Auto-scaling based on load patterns
  • Circuit breaker pattern for fault tolerance

API Reference

RESTful API Documentation

Comprehensive API with OpenAPI 3.0 specification

Blacklist Management

GET /api/blacklist/active

Retrieve all active blacklisted IPs in plain text format for firewall integration

POST /api/blacklist/add

Add new IP addresses to the blacklist with metadata and threat level

DELETE /api/blacklist/{ip}

Remove specific IP address from the blacklist
