🚀 Quick Start

Deploy in production in 15 minutes:

# Online Deployment
docker pull registry.jclee.me/jclee94/blacklist:latest
docker run -d -p 32542:2541 blacklist

# Offline Deployment (Air-gapped)
tar -xzf blacklist-offline-package-v2.0.tar.gz
cd blacklist-offline-package-v2.0
sudo ./scripts/install.sh
